# Custom Enchant Crafting Datapack

A Minecraft datapack that enables players to craft enchanted books through custom recipes, providing an alternative to obtaining enchantments through loot, villagers, or enchanting tables.

---

## Features

* Craft enchanted books using custom recipes
* Supports multiple Minecraft versions (1.21.x series)
* Organized by version folders for streamlined updates
* Includes fixes for enchantment component formats
* Designed to maintain balanced survival progression

---

## Installation

1. Download the latest release.
2. Place the datapack in your world's datapacks folder:

   ```
   /saves/<your world>/datapacks/
   ```
3. Reload the world:

   ```
   /reload
   ```

---

## Usage

* Craft enchanted books using the custom recipes available in-game.
* Apply and combine enchantments using anvils, where supported by the Minecraft version.
* Higher enchantment levels require more advanced crafting recipes.

---

## Notes

* Enchantment data handling may vary between Minecraft versions.
* Ensure you are using the correct version folder for your game version.
* If issues occur, verify that the datapack version matches your Minecraft version.

---

## Known Issues

* Some versions may have incomplete enchantment support.
* Compatibility may vary between snapshot and stable releases.
* There was an issue where you couldn't use books in anvils but that got fixed in update 1.1.1
